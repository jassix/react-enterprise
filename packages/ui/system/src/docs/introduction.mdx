# Lume System

The living catalog for `@lume/foundation`, `@lume/primitives`, and `@lume/blocks`.

## What lives here

- **Foundation** — design tokens, recipes, motion, and the PandaCSS preset.
- **Primitives** — Ark UI components styled with the `@lume/foundation` recipe system.
- **Blocks** — composed, opinionated patterns built from primitives.

## Authoring stories

Colocate `*.stories.tsx` next to each component (same folder as `button.tsx`).
Use CSF3 + the `autodocs` tag to auto-generate prop tables from TypeScript.

```tsx
import type { Meta, StoryObj } from "@storybook/react-vite";
import { Button } from "./button";

const meta = {
  title: "Primitives/Buttons/Button",
  component: Button,
  tags: ["autodocs"],
} satisfies Meta<typeof Button>;

export default meta;
export const Solid: StoryObj<typeof meta> = { args: { variant: "solid" } };
```

## Taxonomy

Mirror the `packages/ui/primitives/src/ui/` folder structure:
`Primitives / Buttons / *`, `Primitives / Overlays / *`, and so on. Blocks use
`Blocks / <Domain> / *`.
